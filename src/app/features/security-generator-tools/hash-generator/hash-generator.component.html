📚 About Hashes

  • Hashes are one-way cryptographic functions.
  • Common algorithms: SHA-256, SHA-1, MD5 (MD5 not supported in browser crypto).
  • Used for data integrity, passwords, digital signatures, and more.
  • Hashes are not reversible and are unique for each input.

📖 How Hashing Works

Hashing transforms your input into a fixed-length string using a mathematical algorithm.
