📚 About UUIDs

  • UUID stands for Universally Unique Identifier.
  • Version 4 UUIDs are randomly generated and highly unlikely to collide.
  • Commonly used for unique keys in databases, distributed systems, and more.
  • Format: 8-4-4-4-12 hexadecimal digits (e.g., xxxxxxxx-xxxx-4xxx-yxxx-xxxxxxxxxxxx)

📖 How UUIDs are Generated

A version 4 UUID is generated using random numbers, following the RFC4122 standard.
